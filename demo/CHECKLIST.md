# Pre-Interview Checklist

## 48 hours before

- [ ] Install latest **Cursor** from https://cursor.com
- [ ] Clone this repo; open as workspace root
- [ ] Run `cd services/questionnaire-api && npm install`
- [ ] Walk through `DEMO_SCRIPT.md` twice—time yourself (target: 4:30 demo + buffer)
- [ ] Read https://trust.cursor.com and skim https://cursor.com/docs
- [ ] Watch one Cursor webinar (pick: Agent or Enterprise) from cursor.com/blog or YouTube

## 24 hours before

- [ ] Rehearse **5-min pitch** from `PITCH_OUTLINE.md` (record yourself)
- [ ] Memorize opening + closing lines
- [ ] Practice Agent prompt for JIRA-4827 until it completes reliably in <2 min
- [ ] Prepare 3 questions to ask their interviewer (see `QA_PREP.md`)
- [ ] Build 3 slides from `slides/PITCH_SLIDES.md` (optional but recommended)

## 1 hour before

- [ ] Restart laptop; close Slack/email
- [ ] Cursor: increase editor font (16–18px), enable light or dark theme for contrast
- [ ] Test screen share: Cursor window + terminal
- [ ] Disable notifications; Privacy Mode **on** if discussing security
- [ ] Open tabs: `ARCHITECTURE.md`, `questionnaires.ts`, `.cursor/rules/responsive-platform.mdc`
- [ ] Copy Agent prompts from `DEMO_SCRIPT.md` into a scratch buffer

## During session

| Block | Time | Material |
|-------|------|----------|
| Pitch | 5 min | `PITCH_OUTLINE.md` / slides |
| Demo | 5 min | `DEMO_SCRIPT.md` |
| Q&A | 5–7 min | `QA_PREP.md` |

## After session

- [ ] Send thank-you email with one specific insight from their answer
- [ ] Offer pilot outline (20–40 seats, 30 days, platform + product squad)
